## 🔄 更新扩展

### 更新步骤：

1. **修改版本号**
   ```json
   // manifest.json
   {
     "version": "1.0.1"  // 递增版本号
   }
   ```

2. **重新构建**
   ```bash
   npm run build
   ```

3. **打包新版本**
   ```bash
   # PowerShell
   Compress-Archive -Path dist\* -DestinationPath matrixlabs-wallet-v1.0.1.zip -Force
   ```

4. **上传到 Chrome Web Store**
   - 登录开发者控制台
   - 选择你的扩展
   - 点击"上传新版本"
   - 上传新的 ZIP 文件
   - 更新版本说明
   - 提交审核

5. **审核通过后自动推送给用户**

### 版本号规则：
- **主版本号**（Major）：重大功能变更或不兼容更新
- **次版本号**（Minor）：新增功能
- **修订号**（Patch）：Bug 修复

例如：
- `1.0.0` → `1.0.1`（Bug 修复）
- `1.0.1` → `1.1.0`（新增功能）
- `1.1.0` → `2.0.0`（重大更新）
